The viewport

The viewport is the visible content of the browser window. Browsers support:

  • vw 1/100th of the viewport width.
  • vh 1/100th of the viewport height.
  • vmin Returns the min value of width or height.
  • vmax Returns the max value of width or height.

Below square always has a site length of 50vmin (= 50/100th of min(height, width) ). In case of a height of 1000px and a witdh of 500px the square site will be 250px.

Using the viewport allows smoothly font-size calculation
