Ems

Ems are relative units based on the font-size. If the font-size is 16px and e.g. a padding of 2em is used, the browser calculates a computed value of 32px.

Use EMs to define font-size

You can use ems to define font-size, when doing it the em-calculation will take the inherited font-size

Following CSS will show it:

body { font-size:16px;} .calculated { font-size: 2em;}
